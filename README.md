# 🤖 AI Smart Data Analyzer

An AI-powered web application that analyzes any dataset (CSV, Excel, JSON) and generates intelligent insights using Google Gemini.

---

## 🚀 Features

- 📂 Upload CSV, Excel, JSON files
- 🤖 Auto dataset detection (Stock, Sales, Reviews, General)
- 📊 AI-powered insights using Gemini
- 📈 Interactive charts (Plotly)
- 📋 Data preview table
- 📉 Data summary (missing values, duplicates, etc.)
- 🎨 Modern SaaS dashboard UI
- 📥 Download processed dataset

---

## 🧠 AI Capabilities

- Sentiment analysis (reviews)
- Stock trend analysis
- Sales performance insights
- Student performance analysis
- General dataset interpretation
- Anomaly detection

---

## 🛠️ Tech Stack

- Python
- Flask
- Pandas
- Plotly
- Google Gemini API
- HTML, CSS, JavaScript
