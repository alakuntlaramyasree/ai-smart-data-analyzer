# 🤖 AI Smart Data Analyzer Pro

An AI-powered data analytics web application built using Flask and Google Gemini AI.  
It automatically detects dataset types, analyzes data, generates insights, and visualizes results using interactive charts.

---

## 🚀 Features

- 📂 Upload CSV, Excel, JSON files
- 🧠 AI-powered analysis using Gemini 2.5 Flash
- 📊 Automatic dataset type detection (Sales, Stock, Reviews, Student, General)
- 📈 Interactive Plotly charts
- 📉 Correlation & Outlier detection
- 📋 Dataset profiling (missing values, stats, quality score)
- 💾 SQLite history tracking system
- 📥 Download analyzed dataset
- 🎨 Modern SaaS-style UI
- ⚡ Fast and automated insights generation

---

## 🧠 AI Capabilities

- Executive Summary generation
- Key insights extraction
- Trend analysis
- Risk detection
- Business recommendations
- Data quality evaluation
